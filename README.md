# Dashboard para estudiantes 2025

Aplicación web desarrollada en React para gestionar materias, notas y tareas escolares.

## Características

- Gestión de materias y calificaciones
- Lista de tareas interactiva
- Panel de estadísticas dinámico
- Persistencia de datos con LocalStorage
- Diseño responsive

## Tecnologías utilizadas

- React
- JavaScript
- CSS
- Vite

## Funcionalidades

### Gestión de materias

- Agregar materias
- Registrar notas
- Eliminar materias
- Cálculo automático del promedio

### Gestión de tareas

- Crear tareas pendientes
- Marcar tareas como completadas
- Eliminar tareas
- Actualización automática del progreso

### Estadísticas

- Promedio general
- Cantidad de materias
- Porcentaje de tareas completadas

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

Instalar dependencias:

```bash
npm install
```

Ejecutar entorno de desarrollo:

```bash
npm run dev
```
